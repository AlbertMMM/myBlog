How to mine Ethereum on Ubuntu 20.04. LTS

2020-11-20 ::Albert
Prerequisites A working installation of Ubuntu 20.04 LTS is required. An Advanced user can install the server version while a neophyte can install the desktop version. The installation process for the Ubuntu Server Edition is slightly different from the Desktop Edition; Ubuntu Server doesn’t have a GUI by default. The guide assumes you have root privileges to the system. This guide also assumes you already have the necessary hardware to mine Ethereum using the ETHASH algorithm.
